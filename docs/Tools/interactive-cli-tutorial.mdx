# Interactive Linux CLI Tutorial

Welcome to the **Interactive Linux CLI Tutorial**. This tool is designed to help you get hands-on experience with basic Unix commands directly in your browser. 

Instead of just reading a manual, you'll be given mission objectives and an emulated Linux terminal. Type the correct command and hit **Enter** to pass the step and advance your progress! There are two core lessons tailored to standard file and directory management.
